Easily add email subscription form to your website using shortcode and widget.

== Description ==

BlossomThemes Email Newsletter allows you to add email subscription form on your website. The plugin works best with BlossonThemes’ themes.

Adding Subscription Form has never been this easy. Choose an email platform, enter your API key, choose an email list, and start getting new subscribers to your website. 

**The plugin supports MailChimp, MailerLite, ConvertKit and GetResponse.**

You can create any number of Newsletter forms and display them using a shortcode and/or a widget.

Once installed, the plugin page will appear on the Admin dashboard.

== Installation ==
**From your WordPress dashboard**

1. Visit 'Plugins > Add New',
2. Search for 'BlossomThemes Email Newsletter' and install the plugin.
3. Activate ‘BlossomThemes Email Newsletter’ from your Plugins page.
