Travel Blogs : A child theme of Di Blog theme.

== Description ==

Travel Blogs is a Responsive, SEO Friendly and Fully Customizable WordPress theme for Blog. it can be use for many types of blog such as travel, personal, technology, fashion, photography, sports, lifestyle, adventure etc. it comes with front page slider to display featured posts. Full control on theme typography and parts of blog. it uses latest version of bootstrap and cross-browser compatible. it is page builder compatible and also have templates for page builder. theme featured : one click demo import, custom logo, custom header image, seven widgets area, blog options, typography options, woocommerce options, page layouts, footer widgets and much more.

== Installation == 

1. In your admin panel, go to Appearance -> Themes and click the 'Add New' button.
2. Type in 'Travel Blogs' in the search form and press the 'Enter' key on your keyboard.
3. Click on 'Install' and then 'Activate' button to use your new theme right away.
4. Navigate to Appearance > Customize in your admin panel and customize the taste.
