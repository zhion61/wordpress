## How to install

* Unzip and then copy the sgdouble directory to /wp-content/themes/ in your WordPress installation. 
* Activate from admin menu. 

## How to use

* Theme options placed at "Appearance >> Customize"
* Widgets can be found at "Appearance >> Widgets"
